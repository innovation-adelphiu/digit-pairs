Digit Pairs

Digit Pairs is a number puzzle where you use each digit from 0 to 9 exactly once to create pairs that add up to the given values. Each puzzle contains five addition problems, each with a fixed number on the right. Your goal is to fill in the boxes on the left.

Three Ways to Play

1. Pencil-and-Paper

You can print out blank puzzles and select a set of fixed numbers from our list of puzzles.

2. Online

There is an online web-based version available: Play Digit Pairs

3. 3D Printed Version

If you have access to a 3D printer, you can create a physical version of the puzzle where you place numbers on tiles. Model files are available on printables.com and thingiverse.com.
